# 향후 게임 메뉴 아이디어

## 좌표 거리 게임 (예정)

- **개요:** 홈에서 쓰는 위도·경도 입력 UX를 **게임** 쪽으로 확장한다.
- **플레이:** 나와 상대방 각각 위도·경도를 입력(또는 붙여넣기)하고, 두 점 사이 **거리(km)** 를 계산해 보여준다. (구면 거리: Haversine 등, 기존 백엔드/프론트 거리 로직 재사용 가능.)
- **재미 요소(후속):** 거리 구간별 문구, 비교(누가 더 가깝다 등), 점수/칭호는 기획 후 결정.
- **배치:** 추후 **게임** 메뉴(예: `/game` 또는 사다리·점메추와 같은 허브)에 전용 화면 또는 서브 메뉴로 연결.

이 파일은 구현 전 아이디어 메모용이다.
